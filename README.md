*   **[Wait or Drill? Shared Decision-making in Adolescents With Stable Osteochondritis Dissecans of the Knee.](https://pubmed.ncbi.nlm.nih.gov/40260663/)** (PMID: 40260663)
    *   本研究通过选择联合分析调查了青少年膝关节稳定型剥脱性骨软骨炎（OCD）初始治疗方案的选择偏好。关键发现显示：474名参与者中，90.8%的家长更倾向早期钻孔手术而非保守治疗，主要考量是降低治疗失败率（46.3%）和避免二次手术（22.4%）。尽管不同性别对恢复时间（女性更重视）和费用（男性更关注）的侧重略有差异，但总体手术偏好不受人口统计学特征显著影响。
